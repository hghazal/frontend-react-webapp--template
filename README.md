<h1 style="text-align:center;">Roller's Frontend Template</h1>
***

## Stack:
- Node.js
- React
- Redux
- **Testing:** Karma, Mocha, Chai, Sinon
- **Style:** ESLint, [`airbnb style guide`](https://github.com/airbnb/javascript)

## Release Workflow:
- Build the code with production settings
```bash
$ npm run build
```
- Run the app server
```bash
$ npm start
```
